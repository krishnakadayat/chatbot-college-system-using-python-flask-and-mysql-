# chatbot-college-system-using-python-flask-and-mysql-
# 🤖 AI Chatbot (Flask + NLP)

An intelligent chatbot system built using Flask and Natural Language Processing (NLP).  
This chatbot can understand user queries and respond like a real assistant.

---

## 🚀 Features

- ✅ Multi-sentence understanding  
- ✅ Smart intent detection (greeting, help, etc.)  
- ✅ Semantic search using Sentence Transformers  
- ✅ Spell correction system  
- ✅ Context-aware responses  
- ✅ Chat history stored in database  

---

## 🧠 Technologies Used

- Python  
- Flask  
- NLTK  
- Sentence-Transformers  
- MySQL  

---

## 💬 Example

**User:**  
> What courses do you offer? Also tell fee  

**Bot:**  
> 🎓 Courses: BSc CSIT, BCA, BIM, BBM  
> 💰 Fee: Rs. 80,000 per year
